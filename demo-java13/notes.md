# What's new in Java 12

* standard vs. `->`
* new keyword `yield` to prevent from falling to the next clause
* `,` separator for multiple cases
* just a preview

## Multiline string literals

* a.k.a. text blocks
* `"""` + *new line*
* indentation trimming algorithm
* normalized line endings
* preview
